# 高度セキュリティ試験問題集

複数のセキュリティ認定試験の問題集を統合したアプリケーションです。

## 対応試験

- **CISSP** (Certified Information Systems Security Professional)
- **CCSP** (Certified Cloud Security Professional)
- **CEH** (Certified Ethical Hacker)

## 機能

- 統合された選択画面で試験を選択
- 各試験専用の問題集アプリへの遷移
- レスポンシブデザイン対応

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ

Vercelにてデプロイ可能です。

```bash
vercel
```

## 構成

- React + Vite
- 既存の個別問題集アプリとの統合
